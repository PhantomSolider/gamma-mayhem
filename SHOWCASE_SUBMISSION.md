# Gamma Mayhem Showcase Submission

Updated for OpenAI Build Week, July 2026.

## Project Title

Gamma Mayhem

## One-Sentence Pitch

Gamma Mayhem is a cosmic retro survival game where players dodge gamma rays, entropy shards, gravity wells, and black holes while collecting Phi Orbs to trigger a shield pulse.

## Project Description

Gamma Mayhem is a lightweight browser game with a cosmic retro style. Players learn the warnings, collect Phi Orbs, and try to survive as the run gets faster.

## Main Features

- Playable canvas arcade game.
- Keyboard controls and touch controls.
- Gamma ray hazard system with clear warning lanes.
- Entropy shard hazard system with gradual pressure.
- Gravity well movement distortion.
- Black hole pull events that arrive later in the run.
- Early Phi Orb collection system.
- Phi Pulse clearing burst.
- Brief Phi Shield feedback after using the pulse.
- Score, integrity, Phi Charge, and wave HUD with high-score tracking.
- Pause, restart, and game over states.
- No external libraries or assets.

## Demo Flow

1. Open the live game.
2. Start a run.
3. Move with WASD or arrow keys.
4. Dodge gamma rays and entropy shards.
5. Collect Phi Orbs.
6. Trigger Phi Pulse with Space.
7. Show gravity wells and black hole pressure.
8. Explain that the game is built with HTML, CSS, JavaScript, and Canvas.
